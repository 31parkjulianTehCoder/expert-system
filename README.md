#Grøwax
Grøwax is an open-source web game that lets you build a sustainable city/apartment complex.

Grøwax has three modes: Easy mode, Hard mode, and Competitive mode.

##Easy mode
In easy mode, you can build an eco-friendly apartment complex. Select the squares on the left, and place your desired tiles on the interface in the middle.

##Hard mode
In hard mode, you can build a sustainable 3D city. Try to balance population with Carbon Footprint and Power Draw!

##Competitive mode
In competitive mode, you build a mini-city. Competitive mode focuses on getting the highest score possible. Try to balance money, population, sustainability, and power draw to get the highest score!
